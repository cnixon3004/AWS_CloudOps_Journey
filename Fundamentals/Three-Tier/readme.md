# Project:🧱 Three-Tier Web Application – Nextwork.org

## 📌 Project Overview
This project is part of my journey to build, document, and deploy a classic three-tier web application on AWS.
Each tier—presentation, logic, and data—is built and configured from scratch, with the goal of gaining a deeper understanding of how each layer communicates securely and efficiently.

## 🛠 AWS Services Used
| Service                                      | Purpose                                                 |
| -------------------------------------------- | ------------------------------------------------------- |
| **Amazon S3**                                | Hosts the static website (HTML, CSS, JS).               |
| **Amazon CloudFront**                        | Delivers cached content globally to reduce latency.     |
| **AWS Identity and Access Management (IAM)** | Controls access via policies and Origin Access Control. |                                                        
| **Amazon DyanmoDB**                          | Database for storing user and app data.                 |
